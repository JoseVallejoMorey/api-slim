

Crear solicitud

PUT : http://localhost:8888/api-slim/public/solicitud/1


Publicar solicitud

PUT : http://localhost:8888/api-slim/public/publicar/1


Descartar solicitud

PUT : http://localhost:8888/api-slim/public/descartar/1

	  http://localhost:8888/api-slim/public/solicitud/111@111.com?page=1&results=3
